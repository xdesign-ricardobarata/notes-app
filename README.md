## Next 13 note app

Basic note app with Next.js 13 and Pocketbase based on [this youtube video](https://www.youtube.com/watch?v=__mSgDEOyv8)

### Setup

1. Clone this repo
2. Run `npm run i`
3. Start Pocketbase wit `./pocketbase serve`
4. Run `npm run dev`
5. Go to `http://localhost:3000/`
6. Play around
